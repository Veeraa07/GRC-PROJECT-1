# AI Governance, Risk & Compliance Framework

## Overview

This project demonstrates Governance, Risk, and Compliance (GRC) practices within a cybersecurity environment. The objective is to identify risks, implement controls, assess compliance requirements, and establish a governance framework aligned with industry standards.

## Project Goals

* Perform Cyber Risk Assessments
* Develop Risk Registers
* Implement Third-Party Risk Management (TPRM)
* Map Controls to ISO 27001
* Align Security Controls with NIST CSF
* Design AI Governance Controls
* Create Compliance Templates and Checklists
